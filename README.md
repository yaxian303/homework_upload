# nodeUpload

## 实现功能
> *  上传doc,zip
> *  自动命名
> *  上传进度提示

## 实现技术

> *  后端用node + express做服务端，实现的文件上传功能；
(http://www.zhangxinxu.com/wordpress/2013/10/understand-domstring-document-formdata-blob-file-arraybuffer/) 成员的Blob，ArrayBuffer，FileReader，FormData，所以不兼容低版本浏览器，建议在移动端上使用。

## 使用方法
